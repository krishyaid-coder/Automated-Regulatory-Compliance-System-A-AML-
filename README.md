# Automated-Regulatory-Compliance-System-A-AML-
The project describes A-AML an intelligent system utilizing RAG and LLMs to streamline regulatory monitoring. Leveraging all-MiniLM, Faiss, and gemini-2.5-flash, it delivers verifiable, cited answers to complex compliance queries. By ensuring outputs are grounded in exact sources, this system mitigates hallucinations, providing auditable precision.
